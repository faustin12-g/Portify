# Hero Images Directory

## Place Your Images Here

Add 3 images to this directory:

1. **hero-1.jpg** or **hero-1.png** - Portfolio hero/about section example
2. **hero-2.jpg** or **hero-2.png** - Projects showcase example  
3. **hero-3.jpg** or **hero-3.png** - Skills/experience section example

### Image Requirements:
- **Format**: JPG, PNG, or WebP
- **Size**: 800x600px or 1200x900px (3:4 or 4:3 ratio)
- **File Size**: Under 200KB each (optimized)
- **Style**: Modern, professional portfolio screenshots or mockups

### Where to Get Images:
- **Unsplash**: https://unsplash.com (search "portfolio", "dashboard", "web design")
- **Pexels**: https://pexels.com (search "portfolio website")
- **Create your own**: Screenshots of portfolio pages or design mockups

The landing page will automatically display these images once you add them!
